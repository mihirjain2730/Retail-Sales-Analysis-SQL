# Retail Sales Analysis using SQL

## Overview

Developed a SQL-based Retail Sales Analysis project to explore sales data and generate business insights through data cleaning, analysis, and reporting.

## Tech Stack

- SQL
- PostgreSQL

## Features

- Database creation and setup
- Data cleaning and validation
- Exploratory Data Analysis (EDA)
- Sales and customer analysis
- Business insight generation

## Key Analysis

- Top customers by sales
- Category-wise performance
- Monthly sales trends
- Customer purchase behavior
- Shift-wise order analysis

## Outcome

Improved practical SQL skills by solving real-world business analysis problems and generating actionable insights.
